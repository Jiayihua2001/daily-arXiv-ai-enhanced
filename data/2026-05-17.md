*Shanker Kumar,Vikas Jindal*

Main category: Property prediction & informatics

TL;DR: The paper proposes a hybrid machine-learning framework for predicting binary alloy mixing enthalpy by combining a small set of physically grounded cluster-expansion descriptors with simple chemical descriptors, achieving accuracy comparable to or better than full cluster expansion with fewer parameters.

Motivation: Mixing enthalpy is central to alloy stability and high-entropy alloy design, but full cluster expansion models can be parameter-heavy and empirical ML models may miss configurational physics. The goal is to build a compact, accurate, and more transferable predictor at the binary subsystem level.

Method: The authors construct a hybrid model using minimal CE-derived configurational features (mainly 1NN and selected short-range multi-site clusters) together with chemical descriptors such as VEC, electronegativity difference, atomic size mismatch, and molar volume. They benchmark five strategies—full CE, restricted CE, configurational-only ML, empirical-only ML, and hybrid ML—across six BCC refractory binary systems using ten ML algorithms and exhaustive feature combinations with cross-validation.

Result: Across the tested binary systems, the hybrid ML approach consistently matches or exceeds full-CE performance, with R² typically above 0.93 and often above 0.98, while using far fewer fitted parameters. It also shows better stability and cross-system consistency than models based only on configurational or only on empirical descriptors.

Conclusion: A compact hybrid descriptor set can capture both configurational physics and chemical trends effectively, providing a robust and scalable foundation for predicting alloy thermodynamics. The validation is limited to binary subsystems, but the framework is designed to extend to ternary and higher-order alloy systems.

Abstract: The enthalpy of mixing is a key thermodynamic quantity governing alloy stability and serves as a foundational input for high-entropy alloy (HEA) design. In this work, we focus on developing and validating an efficient predictive framework at the binary subsystem level, which constitutes the fundamental building blocks of multicomponent HEAs. Effective Hamiltonians based on Cluster Expansion (CE) can reproduce configurational energies from density functional theory (DFT), but they often involve a large number of interaction parameters and exhibit system specific constraints across chemical systems. Conversely, purely empirical machine-learning (ML) models are computationally efficient but may lack explicit configurational physics. To bridge these approaches, we construct a hybrid framework that combines a minimal set of physically grounded CE-derived configurational descriptors, primarily first nearest-neighbor (1NN) interactions and selected short-range multi-site clusters, with compact chemical descriptors including valence electron concentration (VEC), electronegativity difference (Δχ), atomic size mismatch (δ), and molar volume (V). We systematically compare five modeling strategies (full CE, restricted CE, configurational-only ML, empirical-only ML, and hybrid ML) across six BCC refractory binary systems. Ten ML algorithms and exhaustive feature combinations were evaluated using cross-validation. Within the binary systems considered, the hybrid ML approach consistently matches or exceeds full-CE accuracy (R² > 0.93, frequently > 0.98) while employing substantially fewer fitted parameters. It also demonstrates improved stability and cross-system consistency relative to purely configurational or purely empirical models. Although the present validation is limited to binary subsystems, the physically grounded descriptor construction provides a scalable framework that can be systematically extended to ternary and higher-order systems. The results, therefore, establish a robust and computationally efficient foundation for future multicomponent HEA modeling, rather than constituting a direct validation on quaternary random solid solutions.
